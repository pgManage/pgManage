Checkbox

Tag:

<gs-checkbox></gs-checkbox>

Description:

A checkbox, also known as a tickbox is a clickable box that changes between values when clicked. In most cases the values that the checkbox changes between will be "true" and "false" or "-1" and "0". However, there are two other possibilities: "true", "null" and "false" or "-1", "n" and "0".

Attribute [type]:

Two <gs-checkbox> elements with the [type] attribute set. The [type] attribute sets the value type of the checkbox. If there is no attribute then it does boolean values: true, false If the attribute is set to smalling then it uses smallint values: -1, 0 The javascript in this example is so you can see the value when you click on the checkbox.

Attribute [triplestate]:

Six <gs-checkbox> elements with the [value] and [triplestate] attributes set. The [triplestate] attribute adds the value "null" or "n" depending on the type. The javascript in this example is so you can see the value when you click on the checkbox.

Attribute [tabindex=""]:

With the [tabindex=""] attribute, you can control what order your controls are tabbed to.

Attribute [disabled]:

The [disabled] attribute prevents the checkbox from changing.

Attribute [remove-all] and related:

Sometimes you want the remove the rounded corners on a checkbox. This is usually used with grids.

Attributes [inline] and [mini]:

The [inline] attribute forces the checkbox to be the smallest size possible. The mini attribute lessens the height of the button and removes the border.

Attribute [qs=""]:

The name [qs] stands for "querystring". The querystring is the part in the URL after the "?". These checkboxes will get their initial values from the querystring.
