Visibility

Attributes:

show-on-phone
show-on-tablet
show-on-desktop
hide-on-phone
hide-on-tablet
hide-on-desktop
show-xs
show-small
show-medium
show-large
hide-xs
hide-small
hide-medium
hide-large

Description:

These attributes are for making elements appear and disappear depending on screen size.

Ranges:

phone   //             <= 321px
tablet  //>= 321px AND <= 768px
desktop //>= 768px
xs      //             <= 768px
small   //>= 768px AND <= 992px
medium  //>= 992px AND <= 1200px
large   //>= 1200px

Examples:

Device Attributes Example:

In this example we have a list of divs each with one of the hide or show attributes. In each div are some numbers and the name of the attribute that was used on it. The numbers are just to show at what sizes an element would be visible.

  • 1 means visible at phone size
  • 2 means visible at tablet size
  • 3 means visible at desktop size
Page Size Attributes Example:

In this example we have a list of divs each with one of the hide or show attributes. In each div are some numbers and the name of the attribute that was used on it. The numbers are just to show at what sizes an element would be visible.

  • 1 means visible at smaller then small size
  • 2 means visible at small size
  • 3 means visible at medium size
  • 4 means visible at large size
